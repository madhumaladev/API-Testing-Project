# API-Testing-Project
API Testing Project Using POSTMAN: This project demonstrates end-to-end API testing of Users, Resources and Authentication modules using Postman.

##Live API Documentation
Postman Documentation:
https://documenter.getpostman.com/view/52835001/2sBXcHjfNk

1. PROJECT OVERVEIW
   Manual and API Testing performed on User Management Module.

   
2.SCOPE
  -User Creation API
  -Update User API
  -Delete User API
  -Negative Validation Cases
  
3.TOOLS USED
  -POSTMAN
  -Excel
  -REST APIs
  
4.DELIVERABLES
  -Test Plan
  -Test Cases
  -Defect Report
  -RTM
  -Test Summary Report
  -Postman Collection

5.How to Run
  -Import Postman-Collection.json into Postman
  -Set environment variable:
   base_url = https://reqres.in/api
  -Run collection
